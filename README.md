# Cycles Game

A console RPG written in Python for practicing loops, functions, and game mechanics.

## Description

Explore the game world, collect resources, fight enemies, complete quests, and improve your character.

## Features

### Character Stats

* Health
* Mana
* Energy
* Hunger
* Money

### Inventory System

* Food
* Water
* Potions
* Materials
* Weapons
* Armor

### Gameplay Mechanics

* Combat system
* Magic
* Trading
* Quests
* Loot collection
* Random events

## Requirements

```bash
pip install colorama
```

## Running the Game

```bash
python CyclesGame.py
```

## Learning Goals

This project was created to practice:

* `while` loops
* Functions
* Random events
* Inventory management
* Character statistics
* Console game development
